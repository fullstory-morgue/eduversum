Uninstallation of a program

Program Selection

To uninstall a program, select it and uncheck the box next to it.

Click on < Show Changes >.

 

Confirm changes

In the dialog that follows you can once more see the programs that are going to be uninstalled. You can still change the selection by canceling the uninstallation process.

If you want to continue with the uninstallation, then click < Apply Changes >.

 

Passwort Dialog

For an uninstallation you also need "root" rights. The reasons for this are explained in the "Installation" section.

Type the "root" password and click < OK >.

 

Update Dialog

When you uninstall a package you can skip the updating of the package index. Choose "Skip Resynchronisation".

Click on < Next >.

 

Uninstallation Dialog

Now you can see what will be uninstalled. There you can also see which programs/packages will also be uninstalled.

Click on < Next >.

 

Finishing Dialog

When the uninstallation is finished, click on < Close >.
